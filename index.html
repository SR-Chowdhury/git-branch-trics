List of all branches

  • To see local branches, run this code
    git branch
  • To see remote branches, run this code
    git branch -r
  • To see all local and remote branches, run this code
    git branch -a

See What Branch You're On

  • Run this code
    git status

To create a new branch

  • Run this code
    git checkout -b example-branch-name

Switch to a branch in local repository

  • Run this code
    git checkout example-branch-name

Merge a branch

  • Make sure youre in main branch and for this Run this code
    git status
  • Then merge dat that come from another branch to main branch. Run this code
    git merge example-branch-name

To delete a local branch

  • Make sure youre in main branch and for this Run this code
    git status
  • To delete a branch: Run this code
    git branch -d example-branch-name
